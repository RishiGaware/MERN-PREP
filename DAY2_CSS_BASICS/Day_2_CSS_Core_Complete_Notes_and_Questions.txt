1. CSS FUNDAMENTALS
==================================================
What is CSS?
- CSS stands for Cascading Style Sheets
- Used to style HTML, control layout, design, responsiveness, and user experience

Ways to Apply CSS
- Inline CSS (style attribute)
- Internal CSS (<style> tag)
- External CSS (recommended)

      1. Inline CSS:
        <p style="color: red;">Error message</p>

      2. Internal CSS:
        <style>
          p {
            color: red;
          }
        </style>
        or
          <style>
            .login-btn { background: green; color: white; }
          </style>

          <button class="login-btn">Login</button>

      3. External CSS:
        <link rel="stylesheet" href="styles.css">

CSS Syntax
selector {
  property: value;
}

Possible Interview Questions:
- What is CSS and why is it used?
  answer: CSS is used to style HTML, control layout, design, responsiveness, and user experience

- Difference between inline, internal, and external CSS?
  answer: Inline CSS is applied directly to an element using the style attribute. Internal CSS is defined within a <style> tag in the HTML document. External CSS is stored in a separate .css file and linked to the HTML document using the <link> tag.

- Which CSS method is best and why?
  answer: External CSS is recommended as it provides better organization, reusability, and maintainability.

2. CSS SELECTORS
==================================================
Basic Selectors
- Universal selector (*)
  example: * { margin: 0; padding: 0; }

- Element selector (div, p)
  example: div { background-color: #f0f0f0; }

- Class selector (.class)
  example: .btn { background-color: green; color: white; }
  
- ID selector (#id)
  example: #header { background-color: #333; color: white; }

Combinators
- Group selector (h1, h2)
  example: h1, h2 { font-size: 24px; }

- Descendant selector (div p)
  example: div p { color: blue; }

- Child selector (div > p)
  example: div > p { font-weight: bold; }

- Adjacent sibling (h1 + p)
  example: h1 + p { margin-top: 10px; }

- General sibling (h1 ~ p)
  example: h1 ~ p { margin-top: 20px; }

Attribute Selectors
- [type="text"]
  example: input[type="text"] { width: 200px; }

- [disabled]
  example: input:disabled { background-color: #ccc; }

- [class^="btn"]
  example: [class^="btn"] { padding: 10px; }

- [class$="primary"]
  example: [class$="primary"] { color: white; }

Interview Questions:
- What are CSS selectors?
  answer: CSS selectors are used to select and style HTML elements.

- Difference between class and id?
  answer: Class is used to select multiple elements with the same class name, while id is used to select a single element with a unique id.

- What is a descendant selector?
  answer: A descendant selector is used to select elements that are nested within other elements.

- Attribute selector use cases?
  answer: Attribute selectors are used to select elements based on their attributes, such as type, class, or id.

Interview Questions:
- Explain CSS specificity
  answer: CSS specificity is a measure of how specific a selector is in a CSS rule. It is used to determine which style rule takes precedence when multiple rules apply to the same element.

- How is priority decided in CSS?
  answer: Priority is determined by the specificity of the selector, with more specific selectors taking precedence over less specific selectors.

- Why !important should be avoided?
  answer: !important should be avoided as it can make debugging difficult and can lead to unexpected behavior.

4. CSS BOX MODEL
==================================================
Components
- Content
  example: div { width: 200px; height: 200px; padding: 10px; border: 2px solid #000; margin: 20px; }

- Padding
  example: div { padding: 10px; }

- Border
  example: div { border: 2px solid #000; }

- Margin
  example: div { margin: 20px; }

Box Sizing
- content-box (default)
  example: div { box-sizing: content-box; }

- border-box (recommended)
  example: div { box-sizing: border-box; }

Advanced Concepts
- Margin collapsing
- Negative margins

Interview Questions:
- Explain the CSS box model
  answer: The CSS box model is a way of representing an HTML element as a box, with content, padding, border, and margin.

- Difference between margin and padding?
  answer: Margin is the space outside the border, while padding is the space inside the border.

- Why use box-sizing: border-box?
  answer: box-sizing: border-box is used to include padding and border in the element's total width and height.

- What is margin collapsing?
  answer: Margin collapsing is the process where two or more margins combine into a single margin.

  answer: em is a relative unit, while rem is a relative unit.

  answer: rem is best for responsive design.
